# Statistical Comparison

The paired unit is a holdout query group. Across 18 queries, ranker minus first-stage NDCG@10 was -0.04330. A deterministic 1,000-resample query bootstrap gave a 95% interval of [-0.09694, 0.01390]. Three groups improved, five were unchanged and ten worsened. Because the interval includes zero and the point estimate is negative, there is no evidence supporting promotion.
