# Veri Bilimi Portfolyosu

Türkiye Yapay Zeka Akademisi ara ödevini ve planlanan final projesini, mevcut
makine öğrenmesi portfolyosuyla aynı artifact-driven uygulamada sunar.

- Ara Ödev: **Veri Bekleniyor**
- Final Projesi: **Planlandı**

Veri dosyaları lisans/yarışma koşulları doğrulanmadan commit edilmez ve normal
uygulama başlangıcında otomatik indirilmez.
